# UX IMPROVEMENTS

## Core principle
App must be fast and usable on iPhone with one hand.

## Priority areas

### 1. Dashboard
- show today
- show upcoming
- show recent activity
- reduce noise

### 2. Add Activity
- must be fastest flow in app
- minimal steps
- multi-select plants
- quick save

### 3. Plant Detail
- clear status
- activity history
- quick actions

### 4. Calendar
- distinguish planned / done / skipped
- easy overview
- not just visual

## Anti-patterns
- too many inputs
- hidden actions
- desktop-style UI
- decorative UI without function
